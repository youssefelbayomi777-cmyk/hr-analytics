```bash

cd "d:/data analysis/hr-analytics"

# Verify remote is set correctly
git remote -v

# Push to GitHub
git push -u origin main
```

## 🔧 Alternative: Create New Repository

If you want a different repository name:

```bash

git remote remove origin

# Add new remote
git remote add origin https://github.com/youssefelbayomi777-cmyk/new-repo-name.git

# Push
git push -u origin main
```
